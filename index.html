What it does

Five tools, one session

Each tool runs independently. Use one, use all. No account needed.

📄
Resume Optimizer
4-stage AI pipeline — senior recruiter scan, Google XYZ bullet rewrite, ATS simulation, 7-second hiring manager skim test. Optional mock interview with per-answer scoring and a final assessment.
✉️
Cover Letter Generator
Generates a letter that sounds like a real person wrote it — not a template. Bans buzzwords, opens with something specific to the company, closes with confidence. Tone selector included.
💼
LinkedIn Rewriter
Rewrites your headline, About section, and skills list for the role you actually want, not the one you have. Uses the same job description as the optimizer.
🔍
Job Match Finder
Extracts what you're good at from your resume and searches live job listings via the Adzuna API. Country selector. Credentials are server-proxied — never exposed to the browser.
📈
Skill Gap Analyzer
Identifies exactly what's missing between your current profile and the role's requirements. Surfaces specific, free learning resources for each gap. Prioritized by impact.
🎤
Mock Interview Engine
7 questions: one opening, three technical, two behavioural (STAR format), one curveball. Per-answer scoring with a rewritten model answer. Final interview verdict at the end.

Security

Production-hardened

Built with a 13-layer security checklist. Full audit record in the private production repo.

✓
All AI output escaped before innerHTML (34 call sites verified with a real CSS/JS parser)
✓
Per-IP rate limiting + daily cap on all AI-calling endpoints
✓
Full security header suite — CSP, HSTS, X-Frame-Options, Referrer-Policy
✓
Pro-tier key stored in sessionStorage only, auto-cleared after 30 min idle
✓
Pydantic input validation — model allowlists, body size caps, control char sanitisation
✓
Job search API credentials server-proxied — never reach the browser
✓
X-Forwarded-For spoofing fix — last hop trusted, not first
✓
Stateless API — no database, no user data stored anywhere
✓
pip-audit + secret scan runs on every CI pull request
✓
Required consent checkbox on all 5 data-submitting tools
